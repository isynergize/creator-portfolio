Implementing instant search for a static blog without a backend.

## Approach

Client-side filtering with debounced input and keyboard shortcuts for power users.

## Key Features

- Fuzzy matching across title, description, and tags
- Keyboard navigation (up/down arrows, enter to select)
- Command+K shortcut to open
- Highlighted search terms in results
